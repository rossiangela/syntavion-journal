üéØ Abstract

This project presents a machine learning system for diagnosing Parkinson's disease through voice analysis using Random Forest classification. The model achieved 92% overall accuracy and 97% recall for Parkinson's patients, with an AUC-ROC of 0.96. The trained model was successfully converted to TensorFlow Lite for deployment on mobile devices and embedded systems, enabling real-time diagnostic support in clinical settings. The system analyzes 22 voice features including jitter, shimmer, and harmonic-to-noise ratios to distinguish between healthy individuals and those with Parkinson's disease.

üõ†Ô∏è Methodology

The research employed a Random Forest Classifier optimized through GridSearchCV for robust performance on the unbalanced dataset. Key technical aspects include:

  • Algorithm: Random Forest with 100 estimators
  • Optimization: GridSearchCV for hyperparameter tuning
  • Dataset: 195 samples (147 Parkinson's, 48 healthy)
  • Features: 22 voice characteristics including frequency, jitter, and shimmer measures
  • Deployment: TensorFlow Lite conversion for mobile platforms

üéØ Clinical Significance

This work addresses the critical need for early Parkinson's detection through non-invasive methods. Voice analysis offers several advantages:

  • Early Detection: Vocal symptoms often appear before motor symptoms
  • Accessibility: Can be deployed in underserved areas
  • Cost-Effective: No specialized equipment required
  • Remote Monitoring: Enables telemedicine applications
